NSFW Image Classification

A deep learning model that detects whether an image is NSFW and classifies the image into different classes of its nature.

  • Used urlretrieve and python script to download images from text files containing the link to the images
  • Used md5 hashing from Hashlib to find the duplicated images and used TensorFlow decode to detect TensorFlow-supported files
  • Used the concept of transfer learning and the VGG16 model is selected to train the model.
  • A Dataset of 9000 images was used, 80% for training, and 20% for testing and validation
  • ImageDataGenerator is added to increase the amount of data
